# Python学生成绩管理系统 python701

## 项目概述

本项目是一个基于Python的学生成绩管理系统，主要包括学生信息管理、学生成绩管理、教师管理、班级管理等核心功能。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0以上
- MySQL 8.0

## 功能模块

- 学生信息管理
- 学生成绩管理
- 教师管理
- 班级管理
- 考试得分查询
- 修改个人信息
- 修改密码

## 系统角色

- 学生
- 教师
- 管理员

## 目录结构

```
.
├── app
│   ├── models
│   ├── views
│   ├── templates
│   └── ...
├── manage.py
└── ...
```

## 部署步骤

1. 安装Python 3.0以上版本
2. 安装MySQL 8.0
3. 克隆或下载项目代码
4. 使用PyCharm打开项目，配置数据库连接
5. 执行`python manage.py runserver`运行项目
